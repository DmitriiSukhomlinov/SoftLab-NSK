Описание абстрактного класса ILoader.
Абстрактный класс ILoader представляет из себя  интерфейс объекта, предназначенного для обработки входящей видеозаписи и получаения данной видеозаписи покадрово, где каждый кадр представляет из себя массив данных. Класс содержит в себе набор методов, в свою очередь, наследники данного класса будут их реализовывать.
